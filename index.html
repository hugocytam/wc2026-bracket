Path Difficulty Index (PDI)

PDI measures the average Elo strength of a team's projected opponents from R32 through SF.

PDI = (avg_Elo_of_projected_opponents_R32→SF − 1600) / 35

Clamped to [1.0, 10.0]. Thresholds: ≤4.9 green (easy) · 5.0–6.9 amber (competitive) · ≥7.0 red (elite gauntlet)

PDI updates after each round as eliminated teams are replaced by actual opponents. Norway's PDI (7.3 red) reflects IvC + Brazil + England + Argentina path — the hardest path remaining.
