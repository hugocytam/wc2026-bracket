🔎 Data Integrity — Eval Criteria

Every build runs the following checks before publishing. Failures block the file from being presented.

Bracket chain: R32→R16→QF→SF pairings verified against FIFA Annex C (references/bracket-chain.md). Never inferred from match number sequence.

Kickoff times: Sourced from FOX Sports / CBS Sports / ESPN official schedules. Times in ET, converted from venue local time. Never from memory.

Match results: Confirmed from ESPN match reports and BBC Sport. Scorers cross-checked across sources.

Odds: Polymarket implied % via defirate.com. Eliminated teams overridden to 0%.

Team news: Sourced from official national team comms, ESPN, The National, FOX Sports buzz. TBD items explicitly flagged as ⚠️ in the eval banner.

Anti-hallucination: Each team appears max once per day's schedule. Matchups are never inferred from group membership.

R32 visibility: R32 match pairs removed from Bracket tab when tournament reaches R16. Full R32 results available in Insights tab.

Path Difficulty Index (PDI)

PDI = (avg_Elo_of_projected_opponents_R32→SF − 1600) / 35 · Clamped [1.0, 10.0]

≤4.9 green (easy) · 5.0–6.9 amber (competitive) · ≥7.0 red (elite gauntlet)

PDI recalculates after each round using actual opponents. Norway's 7.3 RED reflects IvC + Brazil + England + Argentina — tournament's hardest path. England's PDI dropped 5.8→5.4 when Norway replaced Brazil as projected QF opponent.
