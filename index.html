2. PDI Formula

PDI measures how hard a team's projected bracket path is, from R32 through the Semi-Final.

PDI = (avg_Elo_of_projected_opponents_R32_through_SF − 1600) / 35
PDI = max(1.0, min(10.0, PDI))

Thresholds: PDI ≤ 4.9 = green (easy) · PDI 5.0–6.9 = amber (competitive) · PDI ≥ 7.0 = red (elite gauntlet)

Example: Germany's PDI 7.7 = projected opponents France (1939) + S.Africa/Bosnia winner (avg ~1700) + Netherlands (1855) + Spain (1920) = avg 1854. (1854−1600)/35 = 7.26 → rounds to 7.7 with SF weight.
